## Transaction Trend & Spend Analysis

### Objective
Analyze transaction data to identify spending trends, seasonality, and category-wise performance using Python.

### Tools Used
- Python
- pandas, numpy
- matplotlib

### Key Insights
- Identified monthly spending growth patterns
- Travel and Electronics showed highest contribution to total spend
- Seasonal spikes observed in Q2

### Business Value
Helps financial institutions understand customer spending behavior and optimize marketing and credit strategies.
